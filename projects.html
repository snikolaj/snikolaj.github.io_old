Application for e-commerce webscraping and sorting

This application uses a Python script to webscrape information about the 50 newest PC parts on the two most popular Macedonian e-commerce platforms, and then lists all the information in an Electron.js application. It then uses another custom Python script to turn the cyrillic letters into latin, and then the result is piped into the program through a bash script. The results are then edited and sent to the frontend with NodeJS, where they are again edited with Javascript and displayed on an HTML and CSS frontend.

Technologies used:

  • Python
  • Javascript and NodeJS
  • HTML and CSS
  • Electron.js
Wolfenstein 3D renderer

A renderer based on the classic game Wolfenstein 3D. It is a basic 3D renderer with some game-like elements like collectibles and destructible terrain. It uses raycasting for the 3D rendering algorithm, and is rendered using the Processing3D Java graphics library.

Poetry analyzer

An application that analyzes poetry and displays information like meter, syllable count per line and stress for each syllable. The analysis is done with Prosodic AI implemented in a Python script that is piped into a NodeJS backend where the output is processed and sent to the frontend where it is displayed with HTML and CSS in an Electron.js application.

Technologies used:

  • Python
  • Javascript and NodeJS
  • HTML and CSS
  • Electron.js
